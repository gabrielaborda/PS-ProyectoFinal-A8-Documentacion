Descripción del proyecto

¿Qué hace RehabACV?

RehabACV es una plataforma clínica web diseñada para apoyar la rehabilitación motora de niños y adolescentes que han sufrido un accidente cerebrovascular (ACV). Permite a especialistas configurar, ejecutar y analizar sesiones terapéuticas basadas en videojuego mediante el dispositivo Kinect.

El sistema integra un módulo de juego desarrollado en Unity donde el paciente controla un kayak con el movimiento de su cuerpo. Los datos biométricos capturados por Kinect (centro de masa, inclinación, pitch) se almacenan y visualizan en tiempo real para el especialista.

La plataforma gestiona roles diferenciados para superadministradores, administradores y especialistas, con trazabilidad de auditoría de todas las acciones del sistema.

👤
Gestión de pacientes y especialistas
Fichas clínicas completas, consentimiento informado, historial de sesiones y evolución por paciente.
🎮
Configuración de sesión terapéutica
Parámetros ajustables: velocidad, dificultad, duración. Integración directa con Unity via API REST.
📊
Visualización de resultados Kinect
Gráficas de trayectoria del centro de masa, inclinación lateral y frontal, métricas de estabilidad.
🔒
Autenticación JWT y trazabilidad
Control de acceso por roles, log de auditoría con exportación CSV/PDF y persistencia de sesión segura.

¿Por qué RehabACV para el curso de Pruebas?

Este proyecto fue seleccionado por su complejidad real: combina autenticación con roles, flujos críticos de sesión clínica, integración con hardware externo (Kinect) y un contexto de uso médico donde los errores tienen consecuencias directas en pacientes.

  • ✓
    Múltiples roles de usuario con permisos diferenciados
  • ✓
    Flujos críticos: sesión activa, navegación protegida, cierre automático
  • ✓
    Integración con servicios externos (Unity, Kinect API)
  • ✓
    Dominio médico-clínico: requisitos de calidad exigentes
  • ✓
    Arquitectura full-stack con frontend React y backend FastAPI
